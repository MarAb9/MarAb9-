### 🚀 Featured Projects

#### 🟢 Qrib Lik — AI-Powered Local Services Marketplace
Connects users with trusted professionals nearby. Type a request in **English, French, or Moroccan Darija** ("bghit nsbgh l7it" → painter) and it extracts intent with **Google Gemini**, then ranks nearby providers using **PostGIS** geospatial search and an explainable scoring system.

- AI search pipeline with a deterministic fallback so search never breaks when the AI is down or rate-limited
- Prompt-injection-hardened AI layer — AI is advisory only; deterministic backend rules own every final decision
- Explainable scoring: separate provider-quality "Qrib Score" and request-relevance "Match Score"

**Stack:** Angular 21 (Signals + SSR on Cloudflare Workers) · NestJS · PostgreSQL/PostGIS · Prisma · Google Gemini · Docker
